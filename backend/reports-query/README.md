# Reports Query Service

Microservicio para consultas, filtros y reportes de tickets del sistema ISP.

## Estructura inicial
- TypeScript estricto
- Express
- Vitest para pruebas
- Zod para validaciones

## Comandos
- `npm run dev` — desarrollo
- `npm run build` — compilar
- `npm start` — producción
- `npm test` — pruebas
